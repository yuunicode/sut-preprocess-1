# 1. 적용범위


<p>용선성분중 [Mn], [P]의 관리기준과 노내에서의 Mn, P의 거동 및 제어기술에 대해 적용한다.</p>

# 2. 목적


<ul><li>용선중 [Mn]은 제강공정에서 강종별 최소량의 Mn 규격이 있으므로 고로에서부터 성분규격을 설정하여 관리해야 한다.</li><li>용선중 [P]는 고급강(저린강, 극저린강) 제조시 극소량까지 제어해야 하므로 고로에서부터 성분을 관리하여 제강 예비처리공정의 효율이 상승할 수 있도록 관리해야 한다.</li></ul>

### 4.2.1 Mn 의 환원


<p>고로 장입원료중 Mn Source 는 주로 MnO₂(Pyrolusite), Mn₃O₄(Hausmanite), 3Mn₂O₃, MnSiO₃(Braunite), MnO₂ · nH₂O(Psilomelane) 형태로 장입된다.</p>

<p>이들은 화학친화력이 낮아, 고로조건에서는 쉽게 산소의 일부를 방출하고 MnO 가 된다.</p>

<p>이때 MnO₂ 와 Mn₂O₃ 는 비가역 반응에 의해 Mn₃O₄ 까지 환원한다.</p>

<math display="block">3MnO₂ + 2CO → Mn₃O₄ + 2CO₂ + 101,600kcal</math><img alt="Equation-Block snippet" src="components/a881479912d2818d41a9a26cf2085542_5_img.webp"/>

<math display="block">3Mn₂O₃ + CO → 2Mn₃O₄ + CO₂ + 40,660kcal</math><img alt="Equation-Block snippet" src="components/a881479912d2818d41a9a26cf2085542_6_img.webp"/>

<p>환원된 Mn₃O₄ 는 다시 환원제 C 와 반응하여 MnO 까지 환원 한다.</p>

<math display="block">Mn₃O₄ + CO → 3MnO + CO₂ + 12,4000kcal</math><img alt="Equation-Block snippet" src="components/a881479912d2818d41a9a26cf2085542_8_img.webp"/>

<math display="block">CO₂ + C → 2CO</math><img alt="Equation-Block snippet" src="components/a881479912d2818d41a9a26cf2085542_9_img.webp"/>

<math display="block">Mn₃O₄ + C → 3MnO + CO</math><img alt="Equation-Block snippet" src="components/a881479912d2818d41a9a26cf2085542_10_img.webp"/>

<p>위 반응은 CO-CO₂ 혼합 가스 중 CO의 평형함량이 낮은 곳(700°C에 CO 약 10%)에서 잘 진행된다. 이 환원 조건에서는 Fe₃O₄ 의 환원보다 더 용이하게 일어난다. 그러나 Shaft 상부의 고온 아래에서는 Mn₃O₄ 에서 MnO 을 생성하는 반응에 따라 발생된 CO₂ 는 다시 C 와 반응하여 CO 를 생성시키므로 결국 환원과정은 직접환원으로 진행 한다.</p>

##### 


<math display="block">MnO + CO → Mn + CO₂</math><img alt="Equation-Block snippet" src="components/a881479912d2818d41a9a26cf2085542_13_img.webp"/>

<p>위의 반응이 일어날 수 있는 조건은 700°C에서 가스상의 CO₂ 함량이 0.01%를 넘지 않을 때 만이 된다. 고로 내에서는 700°C에서 이렇게 CO₂ 가 적을 수는 없다. CO₂ 가 이정도 될 수 있는 것은 1000°C 이상의 온도에서 환원된 CO₂ 가 전부 C로 반응 될 때 만이다.</p>

<math display="block">MnO + CO → Mn + CO₂</math><img alt="Equation-Block snippet" src="components/a881479912d2818d41a9a26cf2085542_15_img.webp"/>

<math display="block">CO₂ + C → 2CO</math><img alt="Equation-Block snippet" src="components/a881479912d2818d41a9a26cf2085542_16_img.webp"/>

<math display="block">MnO + C → Mn + CO</math><img alt="Equation-Block snippet" src="components/a881479912d2818d41a9a26cf2085542_17_img.webp"/>

<p>MnO 를 고체 C 으로 환원하기 위해서는 일정온도 조건이 필요하고, 600~700°C에서는 반대방향 조차 진행된다.</p>

<math display="block">Mn + CO → MnO + C + 68,640kcal</math><img alt="Equation-Block snippet" src="components/a881479912d2818d41a9a26cf2085542_19_img.webp"/>

<hr/>

<!-- 페이지번호: 3, 파일명: [Mn] [P] 제어기술 기준 -->
<p>반응을 Mn의 환원 방향으로 향하게 하기 위해서는 보다 높은 온도가 필요하다. 즉 Bosh와 Hearth에서 Mn의 탄화물 Mn₃C가 생성되지만, 이 C가 MnO를 Mn으로 심하게 환원시킨다. Mn₃C 생성은 발열반응으로서 흡열 반응인 Mn의 환원 진행을 쉽게 해준다.</p>

<math display="block">MnO + [(8)/(6)]C → [(1)/(3)]Mn₃C + CO - 64kcal</math><img alt="Equation-Block snippet" src="components/e1e11047bddc49483924e2b6758cef89_2_img.webp"/>

<math display="block">MnO + C → Mn + CO - 68.9kcal</math><img alt="Equation-Block snippet" src="components/e1e11047bddc49483924e2b6758cef89_3_img.webp"/>

<p>500°C 또는 그 이하의 온도에서, Mn₂O₃와 MnO₂는 환원하기 시작하고, Mn₃O₄는 570~1050°C 범위에서 환원이 된다. MnO가 C에 의해 Mn까지 환원이 시작되는 온도가 1100°C 부터이다. Fe가 존재하는 경우 1030°C에서도 이 환원이 시작되지만, 1200~1400°C에서 환원율 15~73%가 일어난다. 고로 조건하에서는 Mn이 선종에 약 60~80%가 들어가고, 극히 일부가 가스에 의해 노외로 방출되고, 나머지는 슬래그 중에 녹는다.</p>

<p>최근, Si이 Mn보다 먼저 환원되고 Mn의 환원은 C에 의하지 않고, Si에 의해 행해진다는 의견이 있다.</p>

<math display="block">2MnO + [Si] → 2Mn + SiO₂ - 32.1 kc</math><img alt="Equation-Block snippet" src="components/e1e11047bddc49483924e2b6758cef89_6_img.webp"/>

<p>따라서, Mn의 환원은 노열이 높을수록 [Si]가 높을수록 많이 발생하며, 이는 슬래그의 영기도와 다음 지수와의 관계에서 설명되어진다.</p>

<math display="block">K_{Mn} = [((Mn)√([Si]))/([Mn])]</math><img alt="Equation-Block snippet" src="components/e1e11047bddc49483924e2b6758cef89_8_img.webp"/>

<p>(Mn) : 슬래그중 Mn의 함량</p>

<p>[Si], [Mn]: 용선중 Si 및 Mn의 함량</p>

<p>슬래그의 영기도가 높게 됨에 따라 K_{Mn}의 값은 감소한다, 즉 용선의 [Si] 함량이 적게되고, 용선의 Mn 함량이 올라가면 슬래그의 Mn이 저하된다. 슬래그의 영기도가 불변하고, K_{Mn}가 불변하면 다음과 같이 된다.</p>

<math display="block">K_{Mn} [(1)/(√([Si]))] = [((Mn))/([Mn])]</math><img alt="Equation-Block snippet" src="components/e1e11047bddc49483924e2b6758cef89_12_img.webp"/>

<hr/>

<!-- 페이지번호: 4, 파일명: [Mn] [P] 제어기술 기준 -->
<p>따라서, 이경우 용선의 Si 함량 √([Si])의 상승은 [Mn]의 증대와 (Mn)의 저하를 일으킨다.</p>

<p>그림 1과 2에서는 [Mn]/(MnO)와 용선온도의 관계를 나타낸다.</p>

<p>즉 온도가 높을 때 MnO의 환원이 잘 일어나고 있다.</p>

<p>즉 용선온도의 상승에 의해 용선중 [Si]의 상승을 초래하고</p>

<p>용선중 [Mn]도 상승한다. 그림 3에서는</p>

## 4.3 노내에서의 P 반응


<p>고로 장입 원료 중 P₂O₅ 형태로 산화철 또는 석탄과 결합해서 (FeO)₃P₂O₅—Vivianite, 또는 (CaO)₃P₂O₅—Apatite로 존재한다.</p>

<p>Vivianite의 H₂ 에 의한 환원은 600°C에서 시작하지만 CO에 의한 환원은 더 높은 온도에서 이루어진다. 환원이 급속히 이루어지는 구간은 H₂ 에 의한 환원은 900~ 1000°C이고, CO에 의한 환원은 100~ 1200°C 구간이다. P가 환원될 때는 (FeO)₃P₂O₅ 에서 P와 Fe로 동시에 환원된다.</p>

<math display="block">2Fe₃(PO₄)₂ + 16C = 3Fe₂P + P + 16CO</math>로 반응하고 P는 용선에 녹는다.<img alt="Equation-Block snippet" src="components/687815a68c7e78c5ac15cc8e99a76527_4_img.webp"/>

<p>Apatite의 환원반응은 C, CO 및 H₂에 의한 환원이 가능하다. 1300℃에서 다음의 반응에 의해 P가 생성된다. (CaO)₃P₂O₅ + 5C = 3CaO + 2P + 5CO - 389100 kcal</p>

<p>P₂O₅의 P가 완전히 환원되는 것은 산소와의 친화력이 비교적 적기 때문이다.</p>

<p>용선중에 환원되는 것에서 부터 고로 노내 에의 P의 관리는 불가능하고, 원료단계에서의 관리기준을 강화하는 것이 필요하다.</p>
